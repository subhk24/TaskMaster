# 🧠 TaskMaster – Smart Task Manager Web App

TaskMaster is a modern, responsive task management web application built using **HTML, CSS, and JavaScript**. It helps users organize daily tasks efficiently with features like task categorization, priority tracking, and real-time updates.


## 🚀 Features

- ➕ Add, edit, and delete tasks
- 📅 Set due dates and reminders
- 🔔 Browser notification support for task alerts
- 📂 Task categorization (Work, Personal, Study)
- 🎯 Priority levels (High, Medium, Low)
- 🔍 Search and filter tasks
- 📊 Progress tracking dashboard
- 🌙 Dark mode support
- 📱 Mobile-friendly responsive design
- 🖱️ Drag and drop task reordering
- 💾 Local storage support (data persists in browser)

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Modern UI + Responsive Design)  
- JavaScript (DOM Manipulation, Local Storage, Notifications API)
